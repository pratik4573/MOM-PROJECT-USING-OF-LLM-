# MOM-PROJECT-USING-OF-LLM-

**MOM Generator using LLM (Open Source)**

# Project Overview

This project is an **LLM-based Minutes of Meeting (MOM) Generator** that automatically converts **audio/video meeting recordings** into **structured meeting minutes.**

The system uses **open-source models** to:

**1.** Transcribe speech from audio/video

**2.** Understand the conversation

**3.** Generate clear and structured Minutes of Meeting

This project is useful for **business meetings, interviews, online classes, and discussions.**

# Features

Upload audio/video files (.mp3, .wav, .m4a, .mp4)

Automatic speech-to-text transcription

LLM-powered MOM generation

Structured output:

Meeting Summary

Key Discussion Points

Decisions Made

Action Items

Simple and interactive Streamlit UI

Fully open-source stack
